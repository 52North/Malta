Event Types

Enable and disable event types by clicking on the buttons. Right click on one type button for advanced options namely selecting only one type or all types.

Feature Attributes

This area shows a list of extreme event features corresponding to the current viewport and filter settings.
  • Double-clicking a row zooms the map to the selected entry.
  • Toggling the "Filter Selection" function hides all unselected entries from further interactions.
  • The columns reflect the extreme events attributes. They can be reordered by drag and drop operations. Clicking them sorts the data by the corresponding attribute.
  • Hovering the cursor over column header reveals a button that opens the attribute context menu with additional options:
    • For most attribute types, a generic filter interface allows to constrain the set of features. Currently filtered column names are shown in bold italic.
    • Columns can get hidden/shown using simple check boxes.
    • Direct setting of sorting direction.
  • Right click on an item to open the event menu with details and report generation for single events, and statistics view and data export options for single or multiple events.
    • For report generation and export of data, pop-ups must be allowed by the browser.
  • Elements without geometry are highlighted in italics.
  • The location of the panel can be changed from bottom to right hand side and the panel can be hidden with the buttons in the top right corner.

Time Picker

Allows to filter extreme event data by time interval. The start and end of the interval of interest can get set by dragging the left and right slider thumb or by clicking on the corresponding dates. The arrows at both ends of the time bar allow to enlarge the interval by one time step.

Background Information

This panel allows to select basic map overlays as well as the base map to use. The overlays can enabled and disabled separately and be reordered by drag and drop. Base layers are always drawn first and only one can be active.

Selections

Provides basic tools to select features (= events), to configure visualisation based on currently selected events, and to export the currently selected features.
Select All Visible
Select all features which are visible in the current map extent.
Box Selection
This function allows to click and drag a rectangle on the map to select the area to zoom to.
Hide Unselected
Focuses on all currently selected features by hiding all others.
Zoom to Selection
Zooms the map to the bounds of all currently selected features.
Export as CSV
Download the attributes and geometries (as well known text, WKT) of all currently selected features as a comma seperated values (CSV) file. Pop-ups must be allowed for this function to work.

Statistics

Provides access to the selection statistics functions. Shows a window with automatically updated statistical values for the current selection of features.
